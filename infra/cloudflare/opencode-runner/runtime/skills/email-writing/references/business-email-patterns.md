# Business Email Patterns

Use this reference for non-cold email situations where clarity, continuity, and tone control matter more than outreach tactics.

## Meeting request

Best when:

- You need to schedule a call or meeting
- The recipient already knows the sender or the business context is legitimate

Core pattern:

1. State the reason for the meeting
2. Explain the desired outcome
3. Propose timing or ask for availability
4. Keep the ask specific

Example shape:

```text
Subject: Meeting request: [topic]

Hi [Name],

I'd like to meet to discuss [topic] so we can [outcome].

Would [option 1] or [option 2] work for you? If not, I can adjust.

Best,
[Name]
```

## Standard follow-up

Best when:

- A prior conversation or email already happened
- You need to move the thread forward

Core pattern:

1. Anchor to the prior exchange
2. State what you need now
3. Make the next step easy

Avoid:

- "Just checking in"
- Vague nudges with no reason to respond

## Reply

Best when:

- The sender is responding inside an active thread
- Continuity matters more than persuasion

Core pattern:

1. Answer the key point first
2. Add any necessary context
3. Close with the next step or confirmation

## Difficult message

Best when:

- The sender must decline, reset expectations, apologize, or deliver unwelcome news

Core pattern:

1. Name the issue directly
2. Give the necessary context without rambling
3. State the decision or boundary clearly
4. Offer the next step when appropriate

Tone guidance:

- Calm beats defensive
- Specific beats vague
- Respectful beats overly warm

## Executive summary email

Best when:

- The recipient wants fast comprehension
- The sender needs to update leadership or a senior stakeholder

Core pattern:

1. Open with the main conclusion
2. List 2-4 critical facts or decisions
3. End with the required action, decision, or watchpoint

Keep it tight. Senior readers should be able to grasp the message in seconds.
