# Style Analytical

## Runtime Label

Analytical Style Guide

## Style Rules

- Prioritize clear logic, explicit claims, and defensible reasoning.
- Use concrete examples, mechanisms, and tradeoffs instead of vague assertions.
- Keep rhetorical tone controlled and professional.
- Prefer precise wording over emotional amplification.
- Avoid motivational fluff and unsupported superlatives.

## Structure Guidance

- Start with the key claim or problem framing.
- Organize sections by argument progression.
- Include practical implications or decisions near the end.
