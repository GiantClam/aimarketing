# Page Structures

Use this reference when you need a reliable section order for website pages.

## Homepage

Recommended flow:

1. Hero
2. What the product or company does
3. Why it matters
4. Key benefits or capabilities
5. Proof or trust signals
6. CTA

## Landing Page

Recommended flow:

1. Hero with offer
2. Problem or desired outcome
3. Value explanation
4. Proof or credibility
5. Objection handling
6. CTA

## Product Or Solution Page

Recommended flow:

1. Product or solution framing
2. Who it is for
3. Main benefits
4. Feature-value sections
5. Proof or differentiators
6. CTA

## Pricing Section

Keep it focused on:

- plan differences
- value framing
- reassurance
- next-step clarity

## CTA Section

Good CTA sections usually include:

- one clear ask
- one short value reminder
- one low-friction next step
