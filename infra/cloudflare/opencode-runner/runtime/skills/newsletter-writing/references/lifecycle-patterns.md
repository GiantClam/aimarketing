# Lifecycle Patterns

Use this reference when a request needs a repeatable structure for newsletters or lifecycle emails.

## Newsletter / Editorial Email

Recommended flow:

1. Subject line
2. Opening angle
3. Main story or update
4. Supporting items or links
5. CTA or reply prompt

## Onboarding / Activation

Recommended flow:

1. What the user can achieve
2. One concrete next step
3. Why it matters
4. Reassurance or support
5. CTA

## Nurture / Education

Recommended flow:

1. One useful insight
2. Why it matters
3. Example or explanation
4. Next step

## Retention / Re-engagement

Recommended flow:

1. Value reminder
2. Relevant update or opportunity
3. Return action
4. Clear CTA
