# Style Story Driven

## Runtime Label

Story-Driven Style Guide

## Style Rules

- Use narrative flow with concrete scenes, actors, and turning points.
- Keep facts intact; do not fabricate details to dramatize.
- Balance emotion with clarity and practical value.
- Use vivid but disciplined language that matches the scenario.
- Keep pacing dynamic with short transitions between moments.

## Structure Guidance

- Open with a concrete moment, tension, or observation.
- Build through challenge and response.
- End with takeaway, implication, or action.
