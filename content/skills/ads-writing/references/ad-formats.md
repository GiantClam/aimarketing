# Ad Formats

Use this reference when you need a stable output pattern for performance copy.

## Search Ad Set

Recommended output:

- Headline set
- Description set
- Optional CTA notes

## Paid Social Ad Set

Recommended output:

- Hook line
- Primary text
- Optional headline
- CTA suggestion

## Angle Variant Set

Recommended output:

- Variant label
- Core angle
- Copy

Keep angle labels easy to compare, such as:

- pain point
- desired outcome
- speed
- proof
- differentiation
